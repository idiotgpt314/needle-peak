# Needle Peak

`Needle Peak` is a standalone browser prototype for a very difficult Celeste-style precision platformer.

## Features

- Tight movement with coyote time, jump buffering, variable jump height, wall slide, wall jump, and an air dash
- Room-based progression with transitions and checkpoints
- Hazards, springs, crumble blocks, dash crystals, moving platforms, berries, and a summit goal
- Timer and death counter for repeat clears
- Mobile touch controls and local best-stat persistence

## Run

Open `index.html` directly in a browser, or use a small static server:

```bash
cd /home/lil-archy4/Work/celeste-clone
python3 -m http.server 8124
```

Then open `http://127.0.0.1:8124`.
